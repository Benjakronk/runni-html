Fagstoff

Hva er events?

Et event er en hendelse i nettleseren – f.eks. at brukeren klikker på en knapp, beveger musen eller skriver på tastaturet. JavaScript kan "lytte" etter disse hendelsene og kjøre kode når de skjer.

Vanlige måter å legge til events

<!-- Direkte i HTML -->
<button onclick="hils()">Klikk meg</button>

<!-- Med addEventListener i JS -->
let knapp = document.getElementById("minKnapp");
knapp.addEventListener("click", hils);

function hils() {
alert("Hei! Du klikket på knappen.");
}

Noen vanlige events

  • click – når du klikker
  • mouseover – når musen går over et element
  • mouseout – når musen forlater elementet
  • keydown – når en tast trykkes ned
  • keyup – når en tast slippes
  • submit – når et skjema sendes inn

Event-objektet

document.addEventListener("keydown", function(event) {
console.log("Du trykket på tasten:", event.key);
});

Interaktiv demo: Events i aksjon

Prøv knappene og skriv i tekstfeltet for å se events i bruk.

Kodeprosjekt: Din egen interaktive boks

Oppgave

  1. Lag en <div> med litt tekst inni
  2. Når man klikker på boksen, skal den endre bakgrunnsfarge
  3. Når musen går over boksen, skal teksten endres
  4. Bonus: Tell antall ganger boksen har blitt klikket

Starterkode

<!DOCTYPE html>
<html>
<head>
<link rel="stylesheet" href="styles.css">
</head>
<body>
<div id="boks">Klikk eller hold musen over meg</div>
<script src="script.js"></script>
</body>
</html>

Fasit

/* script.js */
let boks = document.getElementById("boks");
let teller = 0;

boks.addEventListener("click", function() {
teller++;
boks.style.backgroundColor = "lightblue";
boks.innerText = "Klikket " + teller + " ganger!";
});

boks.addEventListener("mouseover", function() {
boks.style.border = "2px solid red";
});

boks.addEventListener("mouseout", function() {
boks.style.border = "none";
});
